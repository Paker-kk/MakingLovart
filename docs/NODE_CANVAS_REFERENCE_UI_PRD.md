# 节点式 AIGC 画布参考 UI PRD

## 1. 背景与目标

当前节点画布需要进一步贴近参考图的低干扰创作界面：画布主体保持暗色点阵，节点以视觉预览为主，promptbar 只保留生成任务必要参数，左侧与左下角菜单承担全局操作入口。

本 PRD 目标是定义一套可实现、可测试的紧凑型节点画布 UI。设计原则为本地优先、图标优先、文字减负、操作可达。

## 2. 本次明确删除范围

promptbar 中删除以下控件：

- 顶部快捷词：风格、标记、聚焦
- 右上角展开/setting 类按钮
- 摄像机按钮
- 文A 翻译按钮
- 参数滑杆 setting 按钮
- 能量/积分展示：14

删除后 promptbar 不再承担复杂二级配置入口，只保留一次生成所需的最短路径。

## 3. Promptbar 规格

### 3.1 布局

- promptbar 跟随选中节点，位于节点下方。
- promptbar 水平中心与选中节点水平中心对齐。
- promptbar 随画布缩放与平移，不做屏幕 fixed 浮层。
- 默认宽度 800px，圆角 10px，暗色背景，低边框对比。

### 3.2 控件

保留控件：

- 模型选择：默认显示 Lib Nano Pro，支持切换当前节点可用模型。
- 画幅比例：默认 16:9。
- 清晰度：默认 2K。
- 张数：默认 1张，支持 1/2/4。
- 生成按钮：右侧上箭头图标。

输入区：

- placeholder：描述你想生成的画面内容，按 / 呼出指令，@ 引用素材。
- 支持 / 指令菜单与 @ 节点/素材引用。
- 支持拖入素材。

## 4. 节点样式规格

### 4.1 媒体节点

- 图片/视频节点为紧凑视觉卡片。
- 节点内不显示 Upload、Run、Default model、Auto API key、drag 等辅助文字。
- 节点标题以轻量 caption 放在节点上方，例如「图片节点 2」「视频节点 2」。
- 节点主体只显示媒体预览或占位图标。
- 上传入口仅在 hover/selected 时出现小浮标，避免默认状态污染画布。

### 4.2 端口与连线

- 输入/输出端口保持小圆点形态。
- 默认隐藏端口文字标签。
- hover 时端口高亮，拖拽连线时显示可连接/不可连接状态。

## 5. 左侧工具栏规格

参考图左侧为纵向悬浮工具条，共 6 个入口，使用 44px 触控目标。

| 位置 | 图标语义 | 功能名称 | 行为 |
| --- | --- | --- | --- |
| 1 | 加号 | Add node | 在当前视口中心创建默认图片生成节点 |
| 2 | 节点网络 | Open workflow map | 打开/关闭节点库与工作流节点面板 |
| 3 | 素材形状 | Open asset sources | 打开/关闭素材来源面板，默认进入素材页 |
| 4 | 时钟 | Show activity history | 打开运行历史/最近活动面板 |
| 5 | 问号 | Open help | 打开模板/帮助入口，展示使用示例与常见任务 |
| 6 | 耳机 | Open support | 打开支持/诊断入口，用于查看环境、日志与问题反馈 |

视觉要求：

- 工具条固定在画布左侧中部。
- 背景 #262626，边框 #363636。
- 主按钮为浅色底加号，其余为线性图标。
- 中间用一条细分割线隔开创作工具与帮助工具。

## 6. 左下角菜单规格

参考图左下角为紧凑横向悬浮控制条，包含 5 类功能。

| 位置 | 功能名称 | 行为 |
| --- | --- | --- |
| 1 | Toggle overview grid | 显示/隐藏小地图或概览网格 |
| 2 | Open material pins | 打开素材定位/素材钉选面板 |
| 3 | Toggle link mode | 打开节点连接模式或节点库，默认高亮 |
| 4 | Zoom out | 画布缩小 |
| 5 | Zoom readout | 显示当前缩放百分比，点击适配视图 |
| 6 | Zoom in | 画布放大 |

视觉要求：

- 固定在画布左下角。
- 背景与左侧工具栏一致。
- link mode 按钮使用 active 背景。
- 缩放百分比使用等宽数字，避免跳动。

## 7. 状态与交互

- 所有图标按钮必须有 aria-label 与 title。
- hover 状态提高亮度，不改变布局尺寸。
- active 状态使用 #3a3a3a 背景。
- disabled 状态降低透明度，不移除按钮。
- 菜单弹层不得因为鼠标从画布移动到菜单而消失。

## 8. 验收标准

- promptbar 中不出现：风格、标记、聚焦、摄像机、文A、14、展开、setting。
- 选中节点后 promptbar 与节点水平中心对齐。
- 左侧工具栏出现 Add node、Open workflow map、Open asset sources、Show activity history、Open help、Open support。
- 左下角菜单出现 Toggle overview grid、Open material pins、Toggle link mode、Zoom out、Zoom in。
- 图片/视频节点内不出现 Upload、Run、Default model、Auto API key、drag。
- 全量测试与生产构建通过。

## 9. 非目标

- 本 PRD 不引入云同步、市场发布、API Key 管理等大型功能。
- 本 PRD 不重构节点运行引擎。
- 本 PRD 不实现真实客服系统，只定义支持/诊断入口的 UI 与后续功能方向。
